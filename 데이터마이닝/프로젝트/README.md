# 데이터마이닝 프로젝트 
 Data Mining Project
-------------------------
## 주제 
자동차의 종에 따라 분류 및 사고의 규모를 보고 차종을 예측하는 모델 만들기

-----------------------
## 목적
데이터를 다양한 분석 기법을 통해 분석하여 어떤 시간대에 사고가 가장 발생했는지 파악하고, <br>
사상자의 수에 따라 가해 차량의 차종이 어떤 차량인지를 파악하는 것에 목적을 두고 프로젝트를 진행

-------------------------
